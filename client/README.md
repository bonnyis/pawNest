# pawNest

유기동물 보호소 위치제공 및 봉사활동 서비스 연계 웹앱사이트

# 스펙

react / typescript / zustand / vite / tailwind

node 20 이상!

# 폴더구조

app/ // 앱 전역 설정 & 엔트리포인트
pages/ // 라우트 단위 화면 (URL과 매칭)
features/ // 기능(도메인)의 동사 단위 모듈 (사용자가 하는 행동!)
entities // 기능(도메인)의 명사 단위 모듈
widgets/ // 페이지에서 조립되는 공용 UI 단위
shared/ // 전역 공용 리소스 & 작은 단위 UI
